# my-stylus-css
A collection of my Stylus CSS files, to make some websites more usable or more to my liking.

## youtube subscription feed

YouTube decided to remove the option to display the subscription feed as a list, currently only
the grid view is available. To solve this, I created CSS which changes the elements back into
a list. Also I hide the *most recent* section, because it annoyed me as well.

## youtube theater mode

YouTube changed how the theater mode works, it now is more like a lesser version of fullscreen,
rather than its own thing. I tried to change the CSS as best as I could to get the old theater mode back.
